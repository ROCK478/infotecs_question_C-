# infotecs_question_CPP

**Тестовое задание для прохождения стажировки в компанию Инфотекс** 

**Позиция: Junior C++ Developer**

**Дата: Август 2025**

**Результат: Приглашение на собеседование**

**Цель:** Создать библиотеку, приложение и Makefile для Linux на языке C++, которые позволяют пользователю логгировать сообщения в файл по приоритетности с указанием даты сохранения сообщения. 

##Особенности: 
- С++ версия 17
- ООП
- Многопоточность, потокобезопасность
- Обработка исключений, подсказки в случае неверного ввода
- Возможность изменить уровень по умолчанию, ниже приоритета которого сообщения не логгируются

##Инструкция пользователя
1. Скачать репозиторий на linux, используя terminal перейти в папку с репозиторием.
2. Сборка проекта

```bash
make
```
3. Запуск проекта со значениями по умолчанию (log.txt, middle)
```bash
make run
```
4. Приложение можно запускать со своими начальными параметрами
```bash
make run filename.txt low
```
Допустимые уровни приоритета: 1/low, 2/middle, 3/high
